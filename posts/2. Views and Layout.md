# Views and Layout

Mastertech is built on [egui](https://github.com/emilk/egui) with a docking system (`egui_dock`) that lets you arrange every tab exactly how you want. Every "View" is a tab you can drag, stack, split, and resize freely.

> See [[0. Table of Contents]] for a full list of tabs available.

---

## Opening and Closing Tabs

Click the **View** menu in the top-left of the application to toggle any tab on or off. Tabs that are already open will be shown with a checkmark. Tabs that are closed can be re-opened from here at any time.

---

## Moving Tabs

Click and drag a tab's **title bar** to move it. While dragging, drop zones will appear:

### Stack (Browser-style)

Drop the tab onto the **center square** of a destination panel to stack it with existing tabs in that zone — it will become a new tab alongside them, selectable by clicking the tab header.

![[Ex. Docking Tabs.png]]
![[Ex. After Docking Tabs.png]]

### Split (Side-by-side)

Drop the tab onto the **edge arrows** (top/bottom/left/right) of a destination panel to split that panel in half and place your tab in the new section.

## Detachable Windows

A few tabs can be **detached into their own floating window**:

- **[[File Browser]]** — Right-click context menu → *Detach File Browser* / *Attach File Browser*
- **[[Websockets]]** — Right-click context menu → *Detach Websocket Console* / *Attach Websocket Console*

This is useful if you want to keep one of these panels visible while working in other tabs.

---

## Tips

- The layout is **not saved between sessions** yet — it resets to the default when you reopen the app. A saved layout feature is planned.
- If a tab gets lost or accidentally closed, use **View** in the top menu to reopen it.
- Most tabs are lazy-loaded — they don't fetch data until you open them for the first time.
