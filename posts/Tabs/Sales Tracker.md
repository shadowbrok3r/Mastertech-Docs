# Sales Tracker

The Sales Tracker tab provides per-employee sales summaries broken down by **pay period**. It shows order counts, payment totals, and notes for each salesperson — making it easy to track performance and review sales history.

> Shared tab — available in both `Mastertech.exe` and at [master-tech.app](https://master-tech.app). See [[0. Table of Contents]].

---

## What It Shows

The Sales Tracker aggregates data from Prestashop orders and Mastertech tasks to show:

| Column | Description |
| ------ | ----------- |
| **Employee** | Salesperson name / initials |
| **Orders** | Number of orders closed in the period |
| **Payments** | Total payment amount for the period |
| **Notes** | Any notes attached to sales |
| **Pay Period** | The selected date range |

---

## Pay Period Selection

Use the pay period selector to switch between date ranges. The tracker will recalculate totals for the selected period.

---

## Order State Breakdown

Sales can be filtered or viewed by order state (e.g., completed sales vs. in-progress). This gives a clear picture of which orders have fully closed vs. which are still in progress.

---

## Use Cases

- **Daily check-in** — See how you're doing for the current pay period
- **Manager review** — Compare performance across the team
- **Dispute resolution** — Verify which orders are attributed to which salesperson
- **Goal tracking** — Monitor progress toward pay period targets
