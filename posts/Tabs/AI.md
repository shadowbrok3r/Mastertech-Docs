# AI

The AI tab is an **OpenAI-powered chat playground** integrated directly into Mastertech. It supports multi-turn threaded conversations and can assist with technical questions, writing, scripting, and more.

> Shared tab — available in both `Mastertech.exe` and at [master-tech.app](https://master-tech.app). See [[0. Table of Contents]].

---

## Layout

### Top — GPT Panel
The main chat interface showing the active conversation thread. Messages from you and the AI are displayed in a conversation view with syntax highlighting for code blocks.

### Thread Tabs
Multiple conversations can be open at once as tabs within the AI view. Each tab is a separate persistent thread — you can jump between them without losing context.

---

## Features

### Multi-turn Conversations
The AI maintains full context across a conversation — you can ask follow-up questions and refer back to earlier parts of the thread naturally.

### Code Highlighting
Code blocks in responses are syntax-highlighted, making it easy to read and copy scripts, command outputs, or configuration snippets.

### Thread Persistence
Threads are saved — you can close and reopen the app and your conversations will still be there.

---

## Use Cases

- **Quick tech questions** — "What does this registry key do?" / "How do I fix X error?"
- **Writing help** — Drafting customer communication or internal notes
- **Script generation** — Ask the AI to write a PowerShell script for a task
- **Understanding crash dumps** — Paste symbolicated stack output from [[Minidump Analysis]] and ask what it means
- **Training** — Walk through unfamiliar procedures with interactive guidance
