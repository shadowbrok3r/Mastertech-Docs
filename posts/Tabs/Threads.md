# Threads

The Threads tab is an **internal store messaging system** — a simple real-time chat for the team. Use it to communicate with coworkers about tasks, customers, or anything else without leaving Mastertech.

> Shared tab — available in both `Mastertech.exe` and at [master-tech.app](https://master-tech.app). See [[0. Table of Contents]].

---

## Layout

The Threads tab is split into two panels:

### Left — Thread List
A list of conversation threads. Each thread has a title and shows the most recent message preview. Click a thread to open it.

### Right — Message Area
The active conversation. Messages are displayed chronologically with:
- Sender name / initials
- Timestamp
- Message text

A text input at the bottom lets you send a new message to the thread.

---

## How Threads Work

- Threads are **store-scoped** — everyone in your store can see and participate in store threads
- Messages update in real time via the SurrealDB live query system
- Threads persist — you can scroll back to see conversation history

---

## Use Cases

- Coordinate with a salesperson about a specific task or customer without leaving the app
- Leave notes for the next shift
- Quick team announcements (e.g., "system C is ready for pickup")
- Ask coworkers questions while a computer is being worked on
