# Mastertech Documentation

Mastertech is a portable Windows executable and companion web application built for PCL technicians. It automates TUR workflows, remote machine management, inventory tracking, sales reporting, and much more — all from a single dockable, customizable interface.

> Download the latest `.exe` at [master-tech.app](https://master-tech.app)

## Desktop Tabs *(Mastertech.exe only)*

| Tab | Description |
| --- | ----------- |
| [[File Browser]] | Built-in file explorer with copy/paste, rename, and progress tracking |
| [[Scripts]] | One-click Windows tuneup scripts, robocopy tools, and custom script queue |
| [[System Information]] | Hardware summary — CPU, RAM, GPU, motherboard, and drive details |
| [[Resource Monitor]] | Live CPU/RAM/GPU/network charts and process table |
| [[Part Order]] | Special part order form for customer/ticket-linked orders |
| [[Minidump Analysis]] | Windows crash dump (`.dmp`) loader and symbolicated stack viewer |
| [[Websockets]] | Direct WebSocket client console for connecting to remote instances |
| [[Downloads]] | Browse and download Mastertech GitHub release assets |
| [[Bug Tracker]] | Submit bug reports directly to the Mastertech GitHub issue tracker |
| [[Quality Check]] | QC placeholder tab (in development) |

## Shared Tabs *(Available in both the app and on the web)*

| Tab | Description |
| --- | ----------- |
| [[My Tasks]] | Your personal task board |
| [[Store Tasks]] | All open tasks for your store |
| [[Completed Tasks]] | Archive of closed/completed tasks |
| [[Task Audit]] | PrestaShop-linked audit table for service orders |
| [[Sales Tracker]] | Per-technician sales and payment totals by pay period |
| [[KOTH]] | King of the Hill — sales leaderboard vs. all employees |
| [[Inventory]] | Store stock, company inventory, and cost breakdowns |
| [[Threads]] | Store-wide internal chat / message threads |
| [[AI]] | OpenAI-powered chat playground with threaded conversations |
| [[Web Console]] | Remote client manager — shell, file explorer, and TUR tools |
| [[Admin Console]] | Admin-only control center for connected machines and script management |
| [[Database Editor]] | Browse and edit raw database tables |
| [[Query Editor]] | Run raw SurrealDB queries *(admin only)* |
| [[Logs]] | In-app log viewer for runtime diagnostics |
| [[Plugins]] | WASM plugin manager for extending Mastertech functionality |
| [[My Tools]] | Personal virtual filesystem and tool storage |
| [[Create Prestashop Order]] | Multi-step order creation form linked to Prestashop |
