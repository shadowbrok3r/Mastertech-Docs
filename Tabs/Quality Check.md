# Quality Check

The Quality Check (QC) tab is a placeholder for an upcoming quality control workflow. It is currently empty and in active development.

> Desktop tab only. See [[0. Table of Contents]] or [[2. Views and Layout]] for all tabs.

---

## Planned Purpose

The QC tab will be used to document quality check sign-offs for computers that have been serviced and are being prepared for sale or pickup. This is the final step before a machine leaves the shop and is tracked alongside the customer's [[4. Task Boards|Task]].

---

## Current Status

This tab is **not yet implemented**. Opening it shows an empty panel.

QC-related notes can currently be added as text in the [[4. Task Boards|Task]] notes field, or in the recommendations section of the [[3. TUR Sheet]].
