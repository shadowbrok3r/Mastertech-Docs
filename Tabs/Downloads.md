# Downloads

The Downloads tab displays all available releases from the Mastertech GitHub repository. You can browse release notes and download new versions of the executable directly from within the app — no need to visit GitHub or the website separately.

> Desktop tab only. See [[0. Table of Contents]] or [[2. Views and Layout]] for all tabs.

---

## Layout

The Downloads tab shows a table of GitHub releases with:

| Column | Notes |
| ------ | ----- |
| **Release tag / version** | e.g. `v4.7.6` |
| **Release name** | Human-readable release title |
| **Date published** | When the release was posted |
| **Assets** | Downloadable files attached to the release |

Click a row to expand it and see the release notes and download buttons for each asset.

---

## Downloading an Update

1. Find the latest release in the table (typically at the top)
2. Click the asset you want to download (usually `Mastertech.exe` or a zip)
3. The file will download to your machine

> 💡 Mastertech is a portable executable — you can replace the old `.exe` with the new one directly. The app also supports self-update functionality.

---

## When to Check for Updates

- Periodically check this tab for new versions — updates often include bug fixes, new script categories, and UI improvements
- After submitting a [[Bug Tracker|bug report]], check back here to see if a fix was released
- The app version is shown in the title bar
